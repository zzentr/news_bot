# news bot 

Бот которому вы устанавливаете интервал времени, а он через него будет отправлять вам последние новости

## Технологии
- Python
- aiogram
- apscheduler

### Установка
``` bash
git clone <https://github.com/zzentr/news_bot.git>
cd news_bot
python3 -m venv venv
source venv/bin/activate
